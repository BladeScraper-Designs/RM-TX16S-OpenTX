How to upgrade modules please refer to the info below:

For internal module running FrOS
1. Put the FrOS update file(.bin) under [ FIRMWARE] folder of SD card .
2. Power on the radio with [ENT] key pressed.
3. Wait for the finish of upgrading.
4. The radio will shut down after the flashing process and restart with flashing process for iXJT automatically.
 
For internal module running openTX
1. Put the firmware under the folder [FIRMWARE] of sd card.
2. Power on the radio and find the firmware,select it by press [ENT].
3. Select 'Flash internal module ' for internal module and select 'Flash external module' for XJT, wait to end.
